# **App Name**: GyanSetu

## Core Features:

- Offline Learning: Allows students to access lessons, quizzes, and videos even without an active internet connection, synchronizing progress when connectivity is restored.
- Multilingual Support: Provides all learning content (lessons, quizzes, videos, and digital literacy modules) in Punjabi, Hindi, and English.
- Progress Tracking Dashboard: Displays student progress, attendance, and performance metrics, accessible to teachers and administrators. Includes generative AI-powered tool to summarize student performance and highlight areas of improvement.
- Gamified Quizzes: Implements gamification (points, badges, leaderboards) to make learning fun and engaging for students.
- Lightweight Design: Ensures the app is optimized to perform efficiently on low-end smartphones with minimal resource usage.
- Digital Literacy Modules: Offers comprehensive digital literacy modules tailored for rural students, including basic computer skills, safe internet practices, and typing skills.
- Parental Access: Grants parents access to view their child's progress reports and academic performance.

## Style Guidelines:

- Primary color: A vibrant blue (#2E9AFE) evoking trust and knowledge.
- Background color: A light blue (#E0F7FA) to ensure readability and a calm learning environment.
- Accent color: A warm orange (#FFB74D) for interactive elements and CTAs, signaling guidance and support.
- Body and headline font: 'PT Sans', a humanist sans-serif, for its modern yet approachable feel.
- Use clear, intuitive icons, following Material Design principles.
- Clean, uncluttered layout optimized for low-resolution screens.
- Subtle animations to provide feedback on interactions, enhancing the user experience without being distracting.
